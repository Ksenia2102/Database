# Создание базы данных PostgreSQL 

Учебная тестовая база данных postgres с фейковыми данными 

## Библиотеки

 - SQLAlchemy - библиотека для работы с реляционными СУБД
 - csv - работа с файлами в формате csv 
 - alembic - работа с миграциями
 - Faker - генерация фейковый данных для добавления 

 ## Процесс создания и структура 
 - Создала таблицы в models.py
 - Подготовила данные для внесения с таблицы (с помощью Faker) в create_data.py
 - Загрузила данные в базу в loader.py
 - Создала запросы в queries.py для тестирования
